About Me

Hi, Im Ben. I'm a Web designer, Front end Developer, Illustrator and occasional speaker. I like to help companies create engaging and acccesible websites.

I am currently doing my work at Sigma, and have previously helped comanies like Envirofone with improving User Experience.

I'm building this site in public on Github Pages, and you can check back from time to time to see how I'm getting on. If you use the minimise buttons on these content panels you can play my simple point and click game (Beware its not finished!)
